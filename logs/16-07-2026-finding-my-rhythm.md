# Finding My Rhythm

**Date**: 2026-07-16  
**Topic**: Sliding Window, DOM, JavaScript

## What I Learned

The more Sliding Window and Two Pointers problems I solve, the more I enjoy this module. It's probably my favorite part of the DSA roadmap so far. The questions feel less about memorizing tricks and more about figuring out what information is actually worth keeping.

On the JavaScript side, I learned about `map()`, `filter()`, and `reduce()`, but the bigger takeaway was understanding how the DOM is structured. Seeing the hierarchy—from `window` to `document`, then the HTML document, and finally elements, attributes, and text nodes—made DOM manipulation feel much less like magic. It finally clicked that JavaScript is simply navigating and modifying this tree.

I also created an X account to start building in public. It's something I've been thinking about for a while, and I figured it's better to start small than keep waiting for the "right time."

Tomorrow's plan is to go deeper into DOM selectors and continue understanding how JavaScript interacts with web pages.
